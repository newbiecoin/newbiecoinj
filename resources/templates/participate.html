Installation instructions

  • Download and install Java SE Runtime Environment 8 from http://www.oracle.com/technetwork/java/javase/downloads/jre8-downloads-2133155.html.
  • Download the latest Newbiecoin software.
  • Unzip the zip file.
  • Run Newbiecoin.jar.
  • If you are using OS X and your security settings prevent you from opening the jar file, right click on it and choose "Open."

Upgrade instructions

  • When you upgrade, you will need to copy and paste database and wallet files from the old version into the new version. These files can be found in the "resources/db" folder.

Importing your addresses

  • If you are a first time user and would like to import your Bitcoin address into Newbiecoin, simply open the Newbiecoin software and go to the "Wallet" page. From there, you can import your private key in WIF format.
  • If you are using Blockchain.info, go to your Wallet, click on "Import / Export," click "Export Unencrypted," choose "Bitcoin-Qt Format," and copy the "priv" key.
